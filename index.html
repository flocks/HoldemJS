HoldemJS

Computing equities for Holdem Poker in Javascript

What is HoldemJS

HoldemJS is a javascript lib for computing equities for holdem Poker. Available in node and in the browser

var HoldemJS = require('holdemjs');

var Dealer = new HoldemJS();

Dealer.compare("AK", "AQ");

gives you the equities AK : 65%, AQ : 35%.

Syntax

I'm currently working on hands range. In the future, you will be able to compute equities of a whole hand range against another one.

You can specify if the two card are suitted or not :

  • AKs for AK suitted
  • AKo for AK offsuitted
  • AK if you don't care

Example : 

Dealer.compare("AK/TT+", "AA")

TT+ for any pair above the pair of Ten.
